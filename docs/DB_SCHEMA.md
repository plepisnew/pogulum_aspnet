## User Schema

| **Field** | **Type** | **Description** |
| -         | -        | -               |
| `id` | `Integer` | ID that is used to uniquely identify Users. |
| `username` | `String` | User's display name. |
| `password_hash` | `String` | User's password hashed. |
| `created_at` | `Date` | Date of account creation. |
